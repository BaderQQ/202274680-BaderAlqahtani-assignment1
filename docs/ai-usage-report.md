# Tools Used & Use Cases
## ChatGPT

I used ChatGPT to help me structure my portfolio website. It helped me understand how to organize sections (About, Projects, Contact) and how to separate files properly. I also used it to improve the wording of my “About Me” section.

## GitHub Copilot 

I used GitHub Copilot for basic code suggestions and auto-completion while writing HTML and CSS. It helped speed up repetitive coding tasks like writing tags and styling elements. It also helps me to nostruct the readme files.

# Benefits & Challenges
## Benefits

- Helped me understand how to structure a responsive website.

- Saved time when generating basic layout and styling .

- Provided explanations for concepts like Flexbox and media queries.

- Helped improve the clarity of my written content.

## Challenges

- Some generated code was more advanced than needed for a beginner assignment.

- I had to simplify and adjust parts of the code to match the assignment requirments.

- AI sometimes provides extra features that were not required in the assignment.
