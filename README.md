# Personal Portfolio Web Application

## 1. Project Description

This project is a simple and responsive personal portfolio website built using HTML, CSS, and JavaScript.  

The website includes three main sections:

- **About Me** – A short introduction and professional summary.
- **Projects** – Two example projects with descriptions and placeholder images.
- **Contact** – A simple contact form with basic JavaScript interaction (no backend required).

The purpose of this project is to demonstrate fundamental web development skills including page structure, styling, responsiveness, and basic interactivity.

---

## 2. Features

- Responsive design using Flexbox and media queries.
- Clean and simple layout.
- Separate HTML, CSS, and JavaScript files.
- Basic JavaScript form interaction.
- Beginner-friendly structure and code organization.
